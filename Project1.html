LUDIVERS Museum Workshop

As a Project Manager, I designed and built a mock Geiger counter for an exhibition at LUDIVERS from September 2023 to January 2024. This project involved managing the project using Gantt Excel, ensuring adherence to deadlines and budgets, and coordinating teams. I also implemented Arduino coding solutions to integrate interactive features into exhibits.

  • Managed complex timelines and project scopes using Gantt Excel.
  • Designed and implemented interactive features using Arduino technology.
