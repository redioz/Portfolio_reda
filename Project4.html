Tic-Tac-Toe Outcome Prediction

From january 2024 to may 2024, I developed an AI model using machine learning to predict the outcome of Tic-Tac-Toe games. This project involved training the model on a large dataset of game outcomes and features, implementing various machine learning algorithms, and evaluating the model's performance.

  • Applied machine learning techniques to predict game outcomes.
  • Collected and processed a comprehensive dataset of Tic-Tac-Toe game data.
  • Implemented and fine-tuned multiple machine learning algorithms.
